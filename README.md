# Data Engineering ZoomCamp 2026

Personal code and exercises for the [Data Engineering ZoomCamp](https://github.com/DataTalksClub/data-engineering-zoomcamp).

## Modules

| # | Topic | Tools |
|---|---|---|
| 01 | Containerization & Infrastructure | Docker, PostgreSQL, Terraform |
| 02 | Workflow Orchestration | Kestra |

## Setup

Each module has its own `docker-compose.yml`. Copy `.env.example` to `.env`, preencha as credenciais e rode:

```bash
docker compose up -d
```
